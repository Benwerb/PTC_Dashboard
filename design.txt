Goal: Build a dashboard with plotly.js that can be hosted as a public github pages website

Data: Files are hosted publicly at view-source:https://www3.mbari.org/lobo/Data/PTCData/
Parsing out files, there are three lines:
1. header and link to parent directory
2. page break
3. links to data
	A. After the <br><br>, A HREF="/lobo/Data/PTCData/DSL003r2_20260116.csv">DSL003r2_20260116.csv can be parsed. "/lobo/Data/PTCData/DSL003r2_20260116.csv" is the link. "DSL003r2_20260116.csv" is the filename.
	B. Files are separated by <br> and the first file is the most recent

Design:
Website title: PTC Dashboard

top left: "files: " drop down list of all files in the directory. default is the most recent file.

plot: center of the website. takes up most of the page.
xaxis: drop down menu with the list of columns. default DATE TIME
yaxis: drop down menu with the list of columns. default VRS VOLTS
xlabel: column name selected for xaxis
ylabel: column name selected for yaxis
plot title: filename
